# \<pattern-library>

This webcomponent follows the [open-wc](https://github.com/open-wc/open-wc) recommendation.

## Installation
```bash
npm i pattern-library
```

## Usage
```html
<script type="module">
  import 'pattern-library/pattern-library.js';
</script>

<pattern-library></pattern-library>
```
